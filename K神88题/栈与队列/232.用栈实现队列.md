### 思路

栈本身是后进先出的，要实现先进先出，需要两个方向的栈
Python没有内置的栈，可以用list或者deque模拟


```python
class MyQueue:

    def __init__(self):
        # 输入栈：接收push数据
        self.stackA=[]
        # 输出栈：负责pop、peek取出数据
        self.stackB=[]

  

    def push(self, x: int) -> None:
        # 使用输入栈
        self.stackA.append(x)

  

    def _transfer(self):
        if not self.stackB:
            while self.stackA:
                self.stackB.append(self.stackA.pop())

  

    def pop(self) -> int:
        self._transfer()
        return self.stackB.pop()

  

    def peek(self) -> int:
        self._transfer()
        return self.stackB[-1]
```
