### 思路

碰到左括号让对应的右括号入栈，碰到右括号判断是否和栈顶元素相等，必须要相等可以不然顺序就乱掉了

```python
class Solution:

    def isValid(self, s: str) -> bool:

        # 首先如果元素个数不是偶数，直接返回False
        if len(s)%2==1:
            return False

        stack=[]

        for x in s:
            if x=='(':
                stack.append(')')

            elif x=='[':
                stack.append(']')

            elif x=='{':
                stack.append('}')

            # 必须要后弹栈顶元素
            elif not stack  or x!=stack.pop() :
                return False

        if stack:
            return False
        return True
```
