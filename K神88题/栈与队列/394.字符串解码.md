### 思路

递归写法，只写第一个的逻辑

从左边开始，如果元素是字母，则返回字母+后面字符串的递归

否则一定是数字，如果是数字，则找后面范围找到左右括号的平衡点


```python
class Solution:

    def decodeString(self, s: str) -> str:

        # 如果s为空，直接返回
        if not s:
            return s

        # 如果是字母
        if s[0].isalpha():
            return s[0]+self.decodeString(s[1:])

  

        # 否则一定是数字，需要找括号
        i=s.find('[')
        balance=1

        for j in count(i+1):
            if s[j]=='[':
                balance+=1
            if s[j]==']':
                balance-=1
                if balance==0:
                    return int(s[:i])*self.decodeString(s[i+1:j])+self.decodeString(s[j+1:])
```
