### 思路
对于153题，没有重复元素，对于nums[mid]，如果小于等于nums[-1]，说明在右半段，反之在左半段

对于这一题有重复元素，当nums[mid]== nums[r]时，无法确定在左边还是右边
```
设以下两个旋转点值为 0 的示例数组，则当 i=0, j=4 时 m=2 ，两示例结果不同。
示例一 [1,    0,1,1,1] ：旋转点 x=1 ，因此 m=2 在 右排序数组 中。
示例二 [1,1,1,    0,1] ：旋转点 x=3 ，因此 m=2 在 左排序数组 中。
```

可以让j=j-1

```python
class Solution:

    def findMin(self, nums: List[int]) -> int:

        l=0
        r=len(nums)-1

        while l<=r:

            mid=(l+r)//2

            if nums[mid]<nums[r]:
                r=mid

            elif nums[mid]>nums[r]:
                l=mid+1

            else:
                r-=1

        return nums[l]
```
