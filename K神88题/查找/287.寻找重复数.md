### 思路

我们希望每个i对应的num[i]也是i，可以原地交换，让nums[nums[i]]=nums[i]

```python
class Solution:

    def findDuplicate(self, nums: List[int]) -> int:

        i=0

        while i<len(nums):

            if nums[i]==i:
                i+=1
                continue

            if nums[nums[i]]==nums[i]:
                return nums[i]

            idx = nums[i]
            nums[i], nums[idx] = nums[idx], nums[i]

        return -1
```

有n个不同的值，我们可以首先常见n个不同的节点，值为1到n。
那如何构建关系呢？索引其实也是一0到n的列表，也就是说除了0之外索引的值和节点的值是完全重合的，但是排列顺序不同，我们可以让i指向nums[i]，这样，对于重复元素一定存在两个索引指向同一个值，在链表中体现就是有环，这就转换成了找环的问题


```python
class Solution:

    def findDuplicate(self, nums: List[int]) -> int:

        def next(i):
            return nums[i]

        slow=0
        fast=0

        while True:
            slow=next(slow)
            fast=next(next(fast))
            if slow==fast:
                break

        fast=0

        while fast!=slow:
            slow=next(slow)
            fast=next(fast)

        return slow
```
