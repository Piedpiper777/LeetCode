### 思路

不能直接将字符串转换为整数


模拟竖式加法计算

用两个指针从两个字符串的末尾遍历


```python
class Solution:

    def addStrings(self, num1: str, num2: str) -> str:

        res=[]
        carry=0

        i=len(num1)-1
        j=len(num2)-1

        while i>=0 or j>=0 or carry>0:

            if i>=0:
                a=int(num1[i])
            else:
                a=0

            if j>=0:
                b=int(num2[j])
            else:
                b=0

            total=a+b+carry
            carry=total//10
            res.append(str(total%10))

            i-=1
            j-=1

        return "".join(reversed(res))
```
