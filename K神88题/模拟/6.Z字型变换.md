### 思路

其实可以从行的角度出发，就是为每一行添加元素

```python
class Solution:

    def convert(self, s: str, numRows: int) -> str:

        if numRows<2:
            return s

        res=["" for _ in range(numRows)]

        i=0
        flag=1

        for x in s:
            res[i]+=x
            i+=flag

            if i==0 or i==numRows-1:
                flag=-flag

        return "".join(res)
```
