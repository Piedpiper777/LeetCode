### 思路

模拟栈

创建一个空栈
从pushed中往新栈中压入元素
当栈顶元素等于poped的首元素（模拟弹出）时，把栈顶元素弹出

直到栈为空

```python
class Solution:

    def validateStackSequences(self, pushed: List[int], popped: List[int]) -> bool:

        stack=[]
        i=0

        for x in pushed:
            stack.append(x)

            while stack and stack[-1]==popped[i]:
                stack.pop()
                i+=1

        if not stack:
            return True
        return False
```
