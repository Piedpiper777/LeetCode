

可以一个个遍历
```python
class Solution:

    def rotateString(self, s: str, goal: str) -> bool:

        n=len(s)
        j=n-1

        while j>=0:
            s=s[1:n]+s[0]

            if s==goal:
                return True
            j-=1

        return False
```

还有一个思路是，如果s可以通过旋转得到goal，则goal+goal这样的拼接字符串一定包含s

```python
class Solution:
    def rotateString(self, s: str, goal: str) -> bool:
        if len(s)==len(goal) and s in (goal+goal):
            return True
        return False
```
