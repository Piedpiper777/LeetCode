### 思路
定义快慢指针，相遇时再从起点开始一个指针，直到这个新指针再和slow相遇，即是入环点

```python
class Solution:

    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:

        fast,slow=head,head

        while fast and fast.next:
            slow=slow.next
            fast=fast.next.next

            if slow==fast:
                new=head

                while new!=slow:
                    new=new.next
                    slow=slow.next

                return new
```
