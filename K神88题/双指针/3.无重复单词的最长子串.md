### 思路

让一个指针遍历s，在字典中存储{ s[i] : i }，如果键s[i]已经在字典中了，说明重复了，那就需要把后面的指针挪上来

```python
class Solution:

    def lengthOfLongestSubstring(self, s: str) -> int:

        res=0
        n=len(s)
        dic=defaultdict()
        j=-1

        for i in range(n):
            if s[i] in dic:
                j=max(j,dic[s[i]])

            dic[s[i]]=i

            res=max(res,i-j)

        return res
```
