### 思路

用一个双端队列维护一个递减序列，对于右侧的新元素，如果队列末尾元素小于新元素，则一一弹出，保持递减，这样队首的元素就是当前窗口的最大值
对于左侧元素，如果当前的左侧索引已经超过了，则弹出
最后逐个添加队首元素即可

```python
class Solution:

    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:

        deque=collections.deque()
        res=[]

        for i,x in enumerate(nums):

            while deque and nums[deque[-1]]<x:
                deque.pop()

            deque.append(i)
            l=i-k+1

            if deque[0]<l:
                deque.popleft()

            if l>=0:
                res.append(nums[deque[0]])

        return res
```
