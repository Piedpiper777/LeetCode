### 思路

假设A，B的独属长度分别是a、b，共享长度为c，则a+c+b== b+c+a，即指针各自走完一遍再从另一个起点开始走，一定能在交叉点碰到


```python
class Solution:

    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:

  

        idx_a=headA
        idx_b=headB

  

        while idx_a!=idx_b:

            if idx_a:
                idx_a=idx_a.next

            else:
                idx_a=headB

            if idx_b:
                idx_b=idx_b.next

            else:
                idx_b=headA

        return idx_a
```
