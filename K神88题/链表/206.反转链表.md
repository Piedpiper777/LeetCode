### 思路

一个个迭代即可
需要先把后面的节点存起来，因为构建反转关系的过程会断开连接

```python
class Solution:

    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:

        # 从头节点出发
        cur=head

        # 定义初始后节点，当前头节点的后节点是none
        pre=None

        while cur:

            # 需要把后面节点存起来
            tem=cur.next

            # 反转关系
            cur.next=pre

            # 迭代更新
            pre=cur
            cur=tem

        return pre
```
