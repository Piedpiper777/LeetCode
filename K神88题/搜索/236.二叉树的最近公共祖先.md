### 思路

递归，在左子树和右子树中分别找最近公共祖先
1.只有left无right：返回left
2.只有right无left：返回right：
3.都有：返回root
4.都没有，返回

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

  

class Solution:

    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':

        if not root or root==q or root==p:
            return root

        left=self.lowestCommonAncestor(root.left,p,q)
        right=self.lowestCommonAncestor(root.right,p,q)

        if left and right:
            return root

        if left:
            return left

        if right:
            return right

        return
```
