### 思路
236题是二叉树，无中序排列要求
本题是二叉搜索树，存在中序排列规律

区别在于：
- 普通二叉树没有大小规则，只能无脑全树遍历，没有任何办法预判 p、q 在左还是右，**强制递归左右两个孩子**，不管孩子是不是`None`，必然会递归到空节点，必须兜底终止
- 二叉搜索树有天然大小规则，可以**定向递归**，只有确定目标一定存在于左 / 右子树时，才会进入递归调用。 题目保证`p、q一定在树里`，所以我们递归进去的子树**必然能命中节点，不会走到 None**。

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:

    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':

        if p.val>root.val and q.val>root.val:
            return self.lowestCommonAncestor(root.right,p,q)

        if p.val<root.val and q.val<root.val:
            return self.lowestCommonAncestor(root.left,p,q)

        else:
            return root
```
