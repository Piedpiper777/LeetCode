### 思路

其实就是判断是否是有向无环图，如果存在环，则不成立

方法1：建立入度表

首先是遍历所有次序列表，建立邻接字典和入度字典，为所有的后续课程添加入度

然后建立一个队列，把所有入度为0的起始节点添加到队列中

遍历队列中的每一个起始节点，弹出，课程数目-1，让其下一个节点的入度-1，然后再将新的起始节点添加到队列中，如此循环往复，如果最终课程数为0说明无环，否则有环

```python
class Solution:

    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:

        # 遍历次序列表，建立入度字典和邻接字典
        indegrees=[0 for _ in range(numCourses)]
        adjaceny=[[] for _ in range(numCourses)]

        for p in prerequisites:
            indegrees[p[0]]+=1
            adjaceny[p[1]].append(p[0])

        # 建立队列，所有入度为0的起始节点入队
        dq=deque()
        for i in range(len(indegrees)):
            if indegrees[i]==0:
                dq.append(i)

        while dq:

            # 把起始节点弹出，总课程数-1
            pre=dq.popleft()
            numCourses-=1

            # 遍历弹出课程的所有下一个课程，其入度-1，如果入度为0，加入队列
            for cur in adjaceny[pre]:
                indegrees[cur]-=1
                if indegrees[cur]==0:
                    dq.append(cur)

        if not numCourses:

            return True

        return False
```
