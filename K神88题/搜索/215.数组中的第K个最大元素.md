### 思路
O(n)的话就不能排序

粗排序，可以把一个数组粗排序成两部分，左边整体小于右边，但是内部无排序

首先需要先写一个粗排序的函数，随机找一个数，分成两部分，左边小于等于这个数，右边大于等于这个数
返回这个数的索引，表示这个数是在升序粗排列中的位置
在升序排列中，第一个大的数是n-1，则第k个大的数就是n-k
返回该索引后用该索引与n-k比较，如果大于n-k说明目标值在该索引左边，反之在右边

```python
class Solution:

    # 先定义一个粗排列函数
    def rougharr(self,nums,left,right):

        # 先随机找一个索引
        p=randint(left,right)

        tem=nums[p]

        # 让这个索引值和左端点交换一下
        nums[p],nums[left]=nums[left],nums[p]

        i=left+1
        j=right

        while True:

            while i<=j and tem>nums[i]:
                i+=1
            while i<=j and tem<nums[j]:
                j-=1

            if i>=j:
                break

            nums[i],nums[j]=nums[j],nums[i]
            i+=1
            j-=1
            
        nums[left],nums[j]=nums[j],nums[left]

        return j

  

    def findKthLargest(self, nums: List[int], k: int) -> int:

        n=len(nums)
        traget_index=n-k

        left=0
        right=n-1

        while True:
        
            i=self.rougharr(nums,left,right)
            
            if i==traget_index:
                return nums[i]
                
            if i>traget_index:
                right=i-1
                
            else:
                left=i+1
```
