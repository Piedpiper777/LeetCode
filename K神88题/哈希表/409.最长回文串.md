### 思路

对所有的字符串出现次数向下取偶数，如果出现奇数，则加1


```python
class Solution:

    def longestPalindrome(self, s: str) -> int:

        cnt=Counter(s)

        res=0
        odd=0

        for c in cnt.values():
        
            rem=c%2
            res+=c-rem
            if rem==1:
                odd=1
                
        return res+odd
```
