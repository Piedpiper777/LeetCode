### 题目描述
给定一个二叉树 `root` ，返回其最大深度。
二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

### 思路
1. 后续遍历，当前树的最大深度，等于左子树的最大深度和右子树的最大深度的最大值+1
2. BFS层序遍历：一层层遍历，遍历一层，+1
### 代码
1. 后续遍历
```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        return max(self.maxDepth(root.left),self.maxDepth(root.right))+1
```

2. 层序遍历
```python
class Solution:

    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        queue=[root]
        res=0
        while queue:
            tem=[]
            for node in queue:
                if node.left:
                    tem.append(node.left)
                if node.right:
                    tem.append(node.right)
            queue=tem
            res+=1
        return res
```
