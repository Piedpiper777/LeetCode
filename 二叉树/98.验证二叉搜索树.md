### 题目描述
给你一个二叉树的根节点 `root` ，判断其是否是一个有效的二叉搜索树。
**有效** 二叉搜索树定义如下：
- 节点的左子树只包含 **严格小于** 当前节点的数。
- 节点的右子树只包含 **严格大于** 当前节点的数。
- 所有左子树和右子树自身必须也是二叉搜索树。

### 思路
1. 中序遍历
先左，在左中比较，再右
2. 前序遍历
先获取中间节点，然后与左右比较
3. 后序遍历
先获取左右的情况和最值，然后与中间比较
### 代码
1. 中序遍历
```python
class Solution:

    pre=float('-inf')

    def isValidBST(self, root: Optional[TreeNode]) -> bool:

        if not root:
            return True

        if not self.isValidBST(root.left):
            return False

        if root.val<=self.pre:
            return False

        self.pre=root.val

        return self.isValidBST(root.right)
```
2. 前序遍历
```python
class Solution:
    pre=float('-inf')
    def isValidBST(self, root: Optional[TreeNode],left=float('-inf'),right=float('inf')) -> bool:

        if not root:
            return True

        x=root.val

        return left< x < right and self.isValidBST(root.left,left,x) and self.isValidBST(root.right,x,right)
```
3. 后序遍历
```python
class Solution:

    def isValidBST(self, root: Optional[TreeNode]) -> bool:

        def dfs(node):

            # 返回 (子树最小值, 子树最大值, 是否合法)
            if not node:
                # 空节点：min=无穷大，max=-无穷小，标记合法
                return (float('inf'), float('-inf'), True)

            # 后序：先递归左、右
            left_min, left_max, left_ok = dfs(node.left)
            right_min, right_max, right_ok = dfs(node.right)
            x = node.val

            # 1. 先判断左右子树本身是否已经非法
            if not left_ok or not right_ok:
                return (0, 0, False)

            # 2. 校验自定义规则：左全部 < x
            if left_max > x:
                return (0, 0, False)

            # 3. 校验自定义规则：右全部 ≥ x
            if right_min < x:
                return (0, 0, False)

            # 计算当前整棵子树的全局最小、全局最大
            cur_min = min(x,left_min)
            cur_max = max(x,right_max)

            return (cur_min, cur_max, True)

        _, _, valid = dfs(root)

        return valid
```
