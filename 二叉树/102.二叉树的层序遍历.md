### 题目描述
给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

### 思路
1. 我自己的思路
stack存当前层节点，用一个tem存下一层节点，遍历完stack后让stack=tem。但是需要打很多补丁
2. 标准思路：双端队列
左边出，右边进
### 代码
- 我自己第一次写的
```python
class Solution:

    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:

        if not root:
            return []

        stack=[root]
        res=[[root.val]]

        while stack:
            tem=[]
            res.append([])
            for node in stack:
                if node.left:
                    tem.append(node.left)
                    res[-1].append(node.left.val)
                if node.right:
                    tem.append(node.right)
                    res[-1].append(node.right.val)
            if not res[-1]:
                del res[-1]
            stack=tem

        return res
```
- 标准写法（双端队列）
```python
class Solution:

    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []

        res=[]
        queue=collections.deque()
        queue.append(root)

        while queue:
            tem=[]

            for _ in range(len(queue)):
                node=queue.popleft()
                tem.append(node.val)

                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)

            res.append(tem)

        return res
```
