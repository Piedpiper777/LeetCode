### 题目描述
给定一个二叉搜索树的根节点 `root` ，和一个整数 `k` ，请你设计一个算法查找其中第 `k` 小的元素（`k` 从 1 开始计数）。

### 思路
中序遍历，从左下角元素开始，第k个元素
先回顾一下中序遍历
```python
class Solution：
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:

        def dfs(node:Optional[TreeNode]) -> None:
            if not node:
                return

            dfs(node.left)
            ans.append(node.val)
            dfs(node.right)

        ans=[]
        dfs(root)

        return ans
		
```
1. 思路一
中序遍历完成之后，对结果取第k-1个元素。但是每次都需要把整个树都遍历一遍，很慢
2. 思路二
只遍历到k
### 代码
1. 直接在中序遍历的结果上返回
```python
class Solution:

    def kthSmallest(self, root: Optional[TreeNode], k: int) -> int:
    
        def dfs(node:Optional[TreeNode]) -> None:
            if not node:
                return

            dfs(node.left)
            ans.append(node.val)
            dfs(node.right)

        ans=[]
        dfs(root)

        return ans[k-1]
```

   class Solution:

            if not root: return
            dfs(root.left)

            if self.k == 0: return

            if self.k == 0: self.res = root.val

        dfs(root)

        return self.res
