### 题目描述
给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。

### 思路
肯定要用到递归了，前序遍历的结构是[根节点，左子树，右子树]，中序遍历的结构是[左子树，根节点，右子树]
因此preorder[0]是 整个二叉树的根
在中序遍历`inorder`中找到preorder[0]的位置，其左边就是左子树，右边就是右子树
对左右子树执行相同的操作

还有写法二，哈希字典，还是没怎么看懂
### 代码
```python
class Solution:

    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:

        if not preorder:
            return None

        # 1.前序遍历的第一个节点就是整个二叉树的根节点
        root=preorder[0]

        # 2.在中序遍历中找到根节点
        left_size=inorder.index(root)

        # 3.中序遍历中，root左边就是左子树，右边就是右子树
        #    前序遍历中，root紧跟着左子树和右子树，已知左子树大小是left_size
        #    因此，左子树在前序遍历中的范围是preorder[1,left_size+1]，右子树的范围是preorder[left_size+1,最后]
        left = self.buildTree(preorder[1:left_size+1], inorder[0:left_size])
        right = self.buildTree(preorder[left_size+1:], inorder[left_size+1:])

        return TreeNode(root,left,right)
```
