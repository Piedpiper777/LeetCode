### 题目描述
给定一个二叉树的根节点 `root` ，返回 _它的 **中序** 遍历_ 。

### 思路
1. 思路一：递归
但是占用额外空间O(h)，h是树的深度
2. 思路二：迭代栈

3. 思路三：Morris 线索（先留空）

### 代码
1. 递归
```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        def dfs(node:Optional[TreeNode]) -> None:
            if not node:
                return
            dfs(node.left)
            ans.append(node.val)
            dfs(node.right)
        ans=[]
        dfs(root)
        return ans
```
2. 迭代栈
```python
class Solution:

    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        stack=[]
        cur=root
        res=[]
        while cur or stack:
            while cur:
                stack.append(cur)
                cur=cur.left
            cur=stack.pop()
            res.append(cur.val)
            cur=cur.right
        return res
```
