### 题目描述
给定一个二叉树的根节点 `root` ，和一个整数 `targetSum` ，求该二叉树里节点值之和等于 `targetSum` 的 **路径** 的数目。
**路径** 不需要从根节点开始，也不需要在叶子节点结束，但是路径方向必须是向下的（只能从父节点到子节点）。

### 思路
DFS 遍历这棵树，遍历到节点 node 时，假设 node 是路径的终点，那么有多少个起点，满足起点到终点 node 的路径总和恰好等于 targetSum？

### 代码
```python
class Solution:

    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
  
        # 1.初始化一个字典，统计前缀和出现的次数
        # key表示从root节点到当前节点的值之和，value表示该和出现的次数
        cnt=defaultdict(int)

        cnt[0]=1

        ans=0

        # 定义递归函数
        def dfs(node: Optional[TreeNode], s: int) -> None:

            if not node:
                return

            nonlocal ans
  
            s+=node.val # 计算当前的前缀和

            ans+=cnt[s-targetSum] # 计算前面的前缀和中，差值出现的次数

            cnt[s]+=1 # 把当前的前缀和次数统计进去

            dfs(node.left,s)
            dfs(node.right,s)

            cnt[s]-=1

        dfs(root,0)

        return ans
```
