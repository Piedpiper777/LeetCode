### 题目描述
给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

### 思路
1. 思路一
层序遍历，每一层的最后一个节点
回顾一下层序遍历
```python
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []

        res=[]
        queue=collections.deque()
        queue.append(root)

        while queue:
            tem=[]

            for _ in range(len(queue)):
                node=queue.popleft()
                tem.append(node.val)
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            res.append(tem)
        return res
```
但是这里没有将每一层分开，只是按照层的顺序排列

### 代码
1. 层序遍历
```python
class Solution:

    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:

        if not root:
            return []

        ans=[]
        cur=[root]

        while cur:

            ans.append(cur[0].val)
            nxt=[]

            for node in cur:

                if node.right:
                    nxt.append(node.right)

                if node.left:
                    nxt.append(node.left)

            cur=nxt

        return ans
```
