### 题目描述
给你一个由 `'1'`（陆地）和 `'0'`（水）组成的的二维网格，请你计算网格中岛屿的数量。

岛屿总是被水包围，并且每座岛屿只能由水平方向和/或竖直方向上相邻的陆地连接形成。

此外，你可以假设该网格的四条边均被水包围。

### 思路

仍然是递归思想，从陆地一个点出发，递归其上下左右点，递归终止条件是出界或者值不再为1。遍历完成即可加1。
### 代码
```python
class Solution:

    def numIslands(self, grid: List[List[str]]) -> int:

        # 1.确定行数和列数
        m=len(grid)
        n=len(grid[0])

        # 2.定义递归函数

        def dfs(i:int, j:int):

            if i>=m or i<0 or j>=n or j<0 or grid[i][j]!='1':
                return

            grid[i][j]='2' # 标注已经走过了

            dfs(i+1,j)
            dfs(i-1,j)
            dfs(i,j+1)
            dfs(i,j-1)

        # 3.遍历

        res=0

        for i,row in enumerate(grid):
            for j, x in enumerate(row):
                if x=='1':
                    dfs(i,j)
                    res+=1

        return res
```
