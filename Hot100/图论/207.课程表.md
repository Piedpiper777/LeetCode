### 题目描述
你这个学期必须选修 `numCourses` 门课程，记为 `0` 到 `numCourses - 1` 。

在选修某些课程之前需要一些先修课程。 先修课程按数组 `prerequisites` 给出，其中 `prerequisites[i] = [ai, bi]` ，表示如果要学习课程 `ai` 则 **必须** 先学习课程  `bi` 。

- 例如，先修课程对 `[0, 1]` 表示：想要学习课程 `0` ，你需要先完成课程 `1` 。

请你判断是否可能完成所有课程的学习？如果可以，返回 `true` ；否则，返回 `false` 。

### 思路

判断是否存在环，有环则false。
1. 入度
入度：指向该节点的变数
构建初始的入度表和邻接图，邻接图是一个列表，包含了该节点指向的所有节点
将所有入度为0的加入队列，一个个弹出，弹出一个，总课程数减一，该元素邻接的下一元素的入度减一，如果入度也为0了，那就也加入队列。总课程数最后为零，则说明无环，否则有环。
环上的节点永远不可能入队，也就不可能出队，所以如果有环，总课程数一定不会减到零。
2. 三色法
0表示未访问过，1表示正在访问，2表示已经访问过了
走到新节点先标记1，然后遍历其下游节点，递归，如果递归的过程中遇到了1说明出现了环
访问结束后标记2
### 代码
1.入度法
```python
class Solution:

    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:

        # 定义一个空队列
        queue=deque()

        # 定义入度表
        indegrees=[0 for _ in range(numCourses)]

        # 定义邻接图
        adjaceny=[[] for _ in range(numCourses)]

        for cur,pre in prerequisites:

            # 入度+1
            indegrees[cur]+=1

            # 邻接图补全
            adjaceny[pre].append(cur)

        # 将所有入度为0的点入队
        for i in range(len(indegrees)):
            if indegrees[i]==0:
                queue.append(i)

        # 对队列中的元素遍历
        while queue:
            # 弹出一个
            pre=queue.popleft()

            # 课程数-1
            numCourses-=1

            for cur in adjaceny[pre]:
                # 入度-1
                indegrees[cur]-=1

                # 对于新的入度为0的入队
                if indegrees[cur]==0:
                    queue.append(cur)

        # 如果总课程数减到零了，则说明无环
        if numCourses==0:
            return True

        return False
```

2. 三色法
```python
class Solution:

    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        graph=[[] for _ in range(numCourses)]

        for cur,pre in prerequisites:
            graph[pre].append(cur)

        color=[0]*numCourses

        def dfs(node):

            if color[node]==1:
                return False

            if color[node]==2:
                return True

            color[node]=1

            for x in graph[node]:
                if not dfs(x):
                    return False

            color[node]=2

            return True

        for i in range(numCourses):
            if not dfs(i):
                return False

        return True
```
