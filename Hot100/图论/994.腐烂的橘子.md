### 题目描述
在给定的 `m x n` 网格 `grid` 中，每个单元格可以有以下三个值之一：
- 值 `0` 代表空单元格；
- 值 `1` 代表新鲜橘子；
- 值 `2` 代表腐烂的橘子。

每分钟，腐烂的橘子 **周围 4 个方向上相邻** 的新鲜橘子都会腐烂。
返回 _直到单元格中没有新鲜橘子为止所必须经过的最小分钟数。如果不可能，返回 `-1`_ 。

### 思路

首先需要统计一开始的腐烂橘子位置，然后遍历每一个腐烂的橘子，让其上下左右腐烂，然后把新腐烂橘子的位置记录下来，作为新的遍历列表


### 代码
```python
class Solution:

    def orangesRotting(self, grid: List[List[int]]) -> int:

        # 1.统计行数和列数
        m=len(grid)
        n=len(grid[0])

        fresh_cnt=0
        bad_list=[]

        res=0

        # 2.统计最初的腐烂位置和新鲜橘子数量

        for i, row in enumerate(grid):
            for j, x in enumerate(row):
                if x==1:
                    fresh_cnt+=1
                elif x==2:
                    bad_list.append((i,j))

        # 3.当最新腐烂列表和新鲜列表还不为空
        while bad_list and fresh_cnt:

            res+=1
            tem=bad_list
            bad_list=[]
            
            for x, y in tem:
                for i,j in (x+1,y),(x-1,y),(x,y+1),(x,y-1):
                    if 0<=i<m and 0<=j<n and grid[i][j]==1:
                        fresh_cnt-=1
                        grid[i][j]=2
                        bad_list.append((i,j))

        # 4.如果还有fresh_cnt，则说明不能完全腐烂，返回-1

        if fresh_cnt:
            return -1

        return res
```
