### 题目描述
给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

### 思路
1. 思路一：前后缀分解
一根柱子 i 上方能存多少水，由它「左边最高墙」和「右边最高墙」里更矮的那一面决定，再减去柱子自身高度。
因此需要算出左边和右边最高的柱子（包括自身）
2. 思路二
对于两个指针l和r，如果知道l的前缀最大值L_surMax和r的后缀最大值R_preMax，中间的[ l + 1, r - 1]还没有遍历，目前已知L_surMax和R_preMax而不知道L_preMax和R_surMax
分情况讨论：
如果L_surMax<R_preMax，由于L_preMax一定大于R_preMax，所以L_preMax>L_surMax，则l处的接水量就是L_surMax-height[l]
如果L_surMax>R_preMax，由于R_surMax一定大于L_surMax，所以R_surMax>R_preMax，则r处的接水量就是R_preMax-height[r]
然后计算谁，谁动
3. 思路三


### 代码
1. 前后缀分解
```python
class Solution:

    def trap(self, height: List[int]) -> int:
        n=len(height)

        sur_max=[0]*n # 前缀最大值
        pre_max=[0]*n # 后缀最大值
        sur_max[0]=height[0]
        pre_max[n-1]=height[n-1]

        for i in range(1,n):
            sur_max[i]=max(sur_max[i-1],height[i])

        for i in range(n-2,-1,-1):
            pre_max[i]=max(pre_max[i+1],height[i])

        res=0

        for sur,pre,h in zip(sur_max,pre_max,height):
            res+=min(pre,sur)-h

        return res
```
2. 双指针
```python
class Solution:
    def trap(self, height: List[int]) -> int:

        n=len(height)

        l=0
        r=n-1

        surMax=0
        preMax=0

        res=0

        while l<r:
            surMax=max(surMax,height[l])
            preMax=max(preMax,height[r])

            if surMax<preMax:
                res+=surMax-height[l]
                l+=1

            else:
                res+=preMax-height[r]
                r-=1

        return res
```
