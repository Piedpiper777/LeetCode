### 题目描述
给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组**是数组中的一个连续部分。

### 思路
1. 动态规划
以当前元素结尾的最大子数组的和为：以上一个元素结尾的最大子数组的和+当前元素，或者当前元素，取其中的最大值
2. 前缀和
当前元素的前缀和-之前的最小前缀和

### 代码
1. 动态规划
```python
class Solution:

    def maxSubArray(self, nums: List[int]) -> int:

        res=nums[0]
        n=len(nums)
        pre=nums[0]

        for i in range(1,n):
            cur=max(pre+nums[i],nums[i])
            res=max(res,cur)
            pre=cur

        return res
```

2. 前缀和
```python
class Solution:

    def maxSubArray(self, nums: List[int]) -> int:

        n=len(nums)
        pre_min=0
        pre=0
        res=float('-inf')

        for i in range(n):
            pre+=nums[i]
            tem=pre-pre_min
            res=max(res,tem)
            pre_min=min(pre,pre_min)

        return res
```
