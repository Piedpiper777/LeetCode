### 题目描述
给你两个单词 `word1` 和 `word2`， _请返回将 `word1` 转换成 `word2` 所使用的最少操作数_  。

你可以对一个单词进行如下三种操作：

- 插入一个字符
- 删除一个字符
- 替换一个字符

```python
class Solution:

    def minDistance(self, word1: str, word2: str) -> int:

        m=len(word1)
        n=len(word2)

        @cache

        def dfs(i,j):
            if i<0:
                return j+1
            if j<0:
                return i+1
                
            if word1[i]==word2[j]:
                return dfs(i-1,j-1)

            return min(dfs(i-1,j),dfs(i,j-1),dfs(i-1,j-1))+1

        return dfs(m-1,n-1)
```
