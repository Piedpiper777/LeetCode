### 题目描述
一个机器人位于一个 `m x n` 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？


### 思路

动态规划问题
对于从(0,0)点走到(m,n)点可以拆分为从(0,0)点走到(m-1,n)和(m,n-1)的和
同理，可以不断递归下去
边界条件：(0,0)是1，i或j小于0则返回0


### 代码
```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        @cache
        def dfs(i,j):
            if i==0 and j==0:
                return 1
            if i<0 or j<0:
                return 0
            return dfs(i-1,j)+dfs(i,j-1)
        return dfs(m-1,n-1)
```

自下而上
```python
class Solution:

    def uniquePaths(self, m: int, n: int) -> int:

        dp=[[0]*(n) for _ in range(m)]

        for j in range(n):
            dp[0][j] = 1

        # 首列全部1
        for i in range(m):
            dp[i][0] = 1


        for i in range(m-1):
            for j in range(n-1):
                dp[i+1][j+1]=dp[i][j+1]+dp[i+1][j]

        return dp[m-1][n-1]
```

空间优化
```python
class Solution:

    def uniquePaths(self, m: int, n: int) -> int:

        # 第一行
        dp=[1]*n

        # 从第二行开始
        for i in range(1,m):
            for j in range(1,n):
                dp[j]=dp[j]+dp[j-1]

        return dp[n-1]
```
