### 题目描述
给你一个整数数组 `nums` 和一个整数 `k` ，请你返回其中出现频率前 `k` 高的元素。你可以按 **任意顺序** 返回答案。


### 思路
我自己的想法是，现统计出现次数，然后按照次数排序即可，但是时间复杂度可能会退化到O(nlogn)

可以用桶的思想，创建一排桶，其中 buckets[c] 存储出现次数为 c 的元素，遍历 cnt，把出现次数为 c 的元素 x 添加到 buckets[c] 中

倒序遍历 buckets，把 buckets[c] 中的元素加到答案中。

一旦答案的长度等于 k，就立刻返回答案。
### 代码
```python
class Solution:

    def topKFrequent(self, nums: List[int], k: int) -> List[int]:

        cnt=Counter(nums)
        # 每一个bucket都是一个列表，记录出现次数为c的所有不同元素

        buckets=[[] for _ in range(len(nums)+1)] # 0存在，但是用不上，所以要+1

        for num,freq in cnt.items():
            buckets[freq].append(num)
        res=[]

        for bucket in reversed(buckets):
            res+=bucket
            if len(res)==k:
                return res
```
