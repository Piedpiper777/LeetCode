### 题目描述
中位数是有序整数列表中的中间值。如果列表的大小是偶数，则没有中间值，中位数是两个中间值的平均值。
• 例如 arr = [2,3,4] 的中位数是 3 。
• 例如 arr = [2,3] 的中位数是 (2 + 3) / 2 = 2.5 。
实现 MedianFinder 类:
• MedianFinder() 初始化 MedianFinder 对象。
• void addNum(int num) 将数据流中的整数 num 添加到数据结构中。
• double findMedian() 返回到目前为止所有元素的中位数。与实际答案相差 10-5 以内的答案将被接受。


### 思路
中位数的思想
保证 left 的大小和 right 的大小尽量相等。规定：在有奇数个数时，left 比 right 多 1 个数。
保证 left 的所有元素都小于等于 right 的所有元素。

只要时时刻刻满足以上两个要求（满足中位数的定义），我们就可以用 left 中的最大值以及 right 中的最小值计算中位数。

分类讨论：

如果当前 left 的大小和 right 的大小相等：
无论 num 是大是小，都可以先把 num 加到 right 中，然后把 right 的最小值从 right 中去掉，并添加到 left 中。

如果当前 left 比 right 多 1 个数：
无论 num 是大是小，都可以先把 num 加到 left 中，然后把 left 的最大值从 left 中去掉，并添加到 right 中。

但是堆原生只有堆顶最小值的弹出方法，没有最大值的弹出方法


### 代码
```python
class MedianFinder:


    def __init__(self):
        self.left=[]
        self.right=[]

    def heappush_max(self,heap,val):
        """新元素压入最大堆"""
        heapq.heappush(heap,-val)

    def heappushpop_max(self,heap,val):
        """新元素压入最大堆，返回最大值"""
        heapq.heappush(heap,-val)
        return -heapq.heappop(heap)


    def addNum(self, num: int) -> None:
        if len(self.left)==len(self.right):
            # 最小值从右边弹出，用原生的最小堆
            tem=heapq.heappushpop(self.right,num)
            # 压进左边最大堆，用自定义的方法
            self.heappush_max(self.left,tem)

        else:
            # 从左边的最大堆先压进再弹出，用自定义的方法
            tem=self.heappushpop_max(self.left,num)
            # 把左边弹出的最大值压进右边最小堆，用原生的方法
            heapq.heappush(self.right,tem)

  

    def findMedian(self) -> float:
        if len(self.left)==len(self.right):
            return (self.right[0]-self.left[0])/2
        else:
            return -self.left[0]
```
