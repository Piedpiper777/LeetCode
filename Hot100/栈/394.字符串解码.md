### 题目描述
给定一个经过编码的字符串，返回它解码后的字符串。
编码规则为: `k[encoded_string]`，表示其中方括号内部的 `encoded_string` 正好重复 `k` 次。注意 `k` 保证为正整数。
你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。
此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 `k` ，例如不会出现像 `3a` 或 `2[4]` 的输入。
测试用例保证输出的长度不会超过 `105`。

### 思路
递归
对于字符串中的元素，如果元素是字母，则看后面的字符串，否则，一定是数字，则找左括号，用一个变量记录括号，左括号+1，右括号-1

### 代码
```python
class Solution:

    def decodeString(self, s: str) -> str:

        # 如果s为空，直接返回
        if not s:
            return s

  
        # 如果元素是字母，则添加后递归后面的
        if s[0].isalpha():
            return s[0]+self.decodeString(s[1:])

        # 否则一定是数字，那就先找到左括号，然后从左括号的下一个元素开始往下看
        # 用一个变量维护括号情况，碰到左括号+1，右括号-1，==0时需要回过头来处理，并递归后面的
        i=s.find('[')
        balance=1
        for j in count(i+1):
            if s[j]=='[':
                balance+=1
            elif s[j]==']':
                balance-=1
                if balance==0:
                    return int(s[:i])*self.decodeString(s[i+1:j])+self.decodeString(s[j+1:])
```
