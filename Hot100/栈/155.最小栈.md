### 题目描述
设计一个支持 `push` ，`pop` ，`top` 操作，并能在常数时间内检索到最小元素的栈。

实现 `MinStack` 类:

- `MinStack()` 初始化堆栈对象。
- `void push(int value)` 将元素 `value` 推入堆栈。
- `void pop()` 删除堆栈顶部的元素。
- `int top()` 获取堆栈顶部的元素。
- `int getMin()` 获取堆栈中的最小元素。


### 思路
1. 使用辅助栈
同步维护一个辅助栈存储最小值
2. 不开辅助栈
有两种写法
### 代码
1. 使用辅助栈
```python
class MinStack:

    def __init__(self):
        # 维护一个原栈和最小栈
        self.stack=[]
        self.minstack=[]

    def push(self, value: int) -> None:
        # 原栈压入
        self.stack.append(value)
        # 最小栈压入
        if not self.minstack or self.minstack[-1]>=value:
            self.minstack.append(value)

    def pop(self) -> None:
        # 弹出原栈，但是如果最小栈也是这个元素，也要弹出
        if self.stack.pop()==self.minstack[-1]:
            self.minstack.pop()

    def top(self) -> int:
        return self.stack[-1]

    def getMin(self) -> int:

        return self.minstack[-1]
```
2. 不开辅助栈
```python
class MinStack:

    def __init__(self):
        self.stack=[(0,inf)]

    def push(self, value: int) -> None:
        self.stack.append((value,min(value,self.stack[-1][-1])))

    def pop(self) -> None:
        self.stack.pop()

    def top(self) -> int:
        return self.stack[-1][0]

    def getMin(self) -> int:
        return self.stack[-1][-1]
```


```python
class MinStack:

    def __init__(self):
        self.stack=[]
        self.mini=inf

    def push(self, value: int) -> None:
        self.stack.append(value-self.mini)
        self.mini=min(self.mini,value)
        # 如果value比原最小值小，则新最小值就是该元素，栈顶元素小于0


    def pop(self) -> None:
        # 如果栈顶元素小于0，说明入栈时更新了最小值，最小值需要还原一下
        # 如果大于0，说明没更新，最小值不需要操作,self.mini-0
        self.mini-=min(self.stack.pop(),0)

    def top(self) -> int:
        # 如果栈顶元素大于零，说明入栈时没更新元素，只需要再加上最小值即可，是self.mini+stack.pop()
        # 如果栈顶元素小于零，说明入栈时更新了最小值，当前最小值即是原值，是self.mini+0
        return self.mini+max(self.stack[-1],0)

    def getMin(self) -> int:

        return self.mini
```
