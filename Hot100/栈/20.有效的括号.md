### 题目描述
给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

### 思路
如果遍历到左括号，就把对应的右括号压入栈，如果遍历到右括号就把栈顶元素弹出，看是不是对应的右括号

### 代码
```python
class Solution:

    def isValid(self, s: str) -> bool:

        if len(s)%2==1:
            return False

        stack=[]

        for x in s:
            # 处理左括号
            if x=='(':
                stack.append(')')
            elif x=='{':
                stack.append('}')
            elif x=='[':
                stack.append(']')

            # 处理右括号
            # 可能前面没有左括号，则栈为空，直接返回False
            elif not stack or stack.pop()!=x:
                return False

        # 可能没有右括号，则栈不为空
        if stack:
            return False

        return True
```
