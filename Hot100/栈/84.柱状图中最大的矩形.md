### 题目描述
给定 _n_ 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。

求在该柱状图中，能够勾勒出来的矩形的最大面积。


### 思路
找到左右两端离当前元素最近的，比当前元素小的元素，如果左边没有则是-1，右边没有则是n
各自往里面缩一格即是左右边界


### 代码
1. 三次遍历
这里左右需要严格小于heights[i]才可以
```python
class Solution:

    def largestRectangleArea(self, heights: List[int]) -> int:

        n=len(heights)
        left=[-1]*n
        right=[n]*n
        stack=[]

        for i,h in enumerate(heights):
            # 栈是递增栈，栈中的元素都是小于当前元素的
            while stack and h<=heights[stack[-1]]:
                stack.pop()
            if stack:
                left[i]=stack[-1]
            stack.append(i)
        stack.clear()

        for i in range(n-1,-1,-1):
            while stack and heights[i]<=heights[stack[-1]]:
                stack.pop()
            if stack:
                right[i]=stack[-1]
            stack.append(i)

        res=0
        for h,l,r in zip(heights,left,right):
            res=max(res,h*(r-l-1))

        return res
```
2. 两次遍历
如果右边写成小于等于，按照r-l-1的计算方法，如果右边真的存在相等的情况会出错，不过，对于最右边的相同元素，计算结果是正确的
```python
class Solution:

    def largestRectangleArea(self, heights: List[int]) -> int:
        n=len(heights)
        left=[-1]*n
        right=[n]*n
        stack=[]

        for i,h in enumerate(heights):
            # 栈是递增栈，栈中的元素都是小于当前元素的
            while stack and h<=heights[stack[-1]]:
                right[stack.pop()]=i
            if stack:
                left[i]=stack[-1]
            stack.append(i)

        res=0

        for h,l,r in zip(heights,left,right):
            res=max(res,h*(r-l-1))

        return res
```
3. 一次遍历
如果当前元素小于等于栈顶元素，那当前元素就是栈顶元素的right，而栈顶元素的left是栈中其前面的一个元素，因此，视角可以说栈顶元素，这样一次遍历即可
但是这样理解起来有点不太容易，所以可以先掌握前两种
```python
class Solution:

    def largestRectangleArea(self, heights: List[int]) -> int:

        heights.append(-1)
        n=len(heights)
        stack=[-1]
        res=0

        for right,h in enumerate(heights):
            while len(stack)>1 and h<=heights[stack[-1]]:
                i=stack.pop()
                left=stack[-1]
                res=max(res,heights[i]*(right-left-1))

            stack.append(right)

        return res
```
