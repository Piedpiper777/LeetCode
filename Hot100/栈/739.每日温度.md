### 题目描述
给定一个整数数组 `temperatures` ，表示每天的温度，返回一个数组 `answer` ，其中 `answer[i]` 是指对于第 `i` 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 `0` 来代替。

### 思路
从左往右的思想
用一个栈作为待处理清单，递减排序
如果当前温度大于栈顶温度，则出栈，一直循环，直到栈顶元素不再小于当前温度

### 代码
```python
class Solution:

    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        n=len(temperatures)
        res=[0]*n
        stack=[]

        for i,t in enumerate(temperatures):
            # 当前栈不为空，说明有需要处理的元素
            # 如果当前元素大于栈顶温度，则说明这栈顶元素找到了答案，出栈
            while stack and t>temperatures[stack[-1]]:
                j=stack.pop()
                res[j]=i-j

            # 如果栈为空，直接入栈
            # 如果栈不为空，处理完比它小的元素后入栈
            stack.append(i)
        return res
```
