### 题目描述
给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。
子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列。

### 思路

表示以nums[i]结尾的最大递增子序列，枚举从0到i-1，对每一个小于nums[i]的数，计算其最大递增子序列，最大一个+1即是以nums[i]结尾的最大递增子序列


还有更优的方法，但是暂时看不懂

现在似乎看懂了
维护一个数组g，g[i]表示长度为i+1的递增子序列的最小末尾元素
如果x严格大于g[-1]则直接把x加进去
如果x不严格大于g[-1]则用二分法找到x的插入位置
### 代码
1. 自上而下
```python
class Solution:

    def lengthOfLIS(self, nums: List[int]) -> int:

        @cache
        def dfs(i):
            res=0

            for j in range(i):
                if nums[j]<nums[i]:
                    res=max(res,dfs(j))

            return res+1

        return max(dfs(i) for i in range(len(nums)))
```

2. 自下而上
```python
class Solution:

    def lengthOfLIS(self, nums: List[int]) -> int:
        f=[0]*len(nums)

        for i, x in enumerate(nums):

            for j, y in enumerate(nums[:i]):
                if y<x:
                    f[i]=max(f[i],f[j])
            f[i]+=1

        return max(f)
```

3. 优化方法
```python
class Solution:

    def lengthOfLIS(self, nums: List[int]) -> int:

        g = []

        for x in nums:
            if not g:
                # 列表为空，直接添加第一个数
                g.append(x)

            elif x > g[-1]:
                g.append(x)

            else:
                j = bisect.bisect_left(g, x)
                g[j] = x

        return len(g)
```
