### 题目描述
给定一个非负整数 _`numRows`，_生成「杨辉三角」的前 _`numRows`_ 行。

在**「杨辉三角」**中，每个数是它左上方和右上方的数的和。


### 思路
左对齐之后的规律：
每一行的首位元素是1
中间元素是上一行的该位置和上一行的前一个位置
即c [i]  [j]=c [i-1]  [j-1]+c [i-1]  [j]


### 代码
```python
class Solution:

    def generate(self, numRows: int) -> List[List[int]]:

        c=[[1]*(i+1) for i in range(numRows)]

        # 从第三行开始

        for i in range(2,numRows):
            # 对每一行的[1:i]
            for j in range(1,i):
                c[i][j]=c[i-1][j-1]+c[i-1][j]
        return c
```
