### 题目描述
给你一个整数数组 `coins` ，表示不同面额的硬币；以及一个整数 `amount` ，表示总金额。

计算并返回可以凑成总金额所需的 **最少的硬币个数** 。如果没有任何一种硬币组合能组成总金额，返回 `-1` 。

你可以认为每种硬币的数量是无限的。

```python
class Solution:

    def coinChange(self, coins: List[int], amount: int) -> int:
	    
	    @cache
        def dfs(i,a):

            # 其中i表示有0-i种面额可选，a表示总额度
            # 边界条件，考虑i和a两个参数,i表示硬币面额索引，因此i==0是有意义的
            # 如果i<0，而a不为0则不合法
            # 如果i<0，a==0，则直接返回0

            if i<0:
                if a==0:
                    return 0
                else:
                    return inf

            # 考虑当前剩余总额度和i面额的大小

            if a<coins[i]:
                # 只能不选i
                return dfs(i-1,a)

            # 选还是不选
            # 选的话是dfs(i,a-coins[i])+1，不选的还是dfs(i-1,a)
            return min(dfs(i,a-coins[i])+1,dfs(i-1,a))
        res=dfs(len(coins)-1,amount)

        if res<inf:
            return res
        return -1
```

2. 自底而上
```python
class Solution:

    def numSquares(self, n: int) -> int:

        """
        阶段：当前要处理的完全平方数
        状态变量：当前要处理的完全平方数和剩余总数
        决策变量：当前要处理的完全平方数是选还是不选
        边界条件：完全平方数为0
        """
        dp=[[inf]*(n+1) for _ in range(isqrt(n)+1)]
        dp[0][0]=0
        
        for i in range(1,isqrt(n)+1):
            s=i*i
            for j in range(n+1):
                dp[i][j]=dp[i-1][j]
                if j>=s:
                    dp[i][j]=min(dp[i][j],dp[i][j-s]+1)
        res=dp[isqrt(n)][n]
        return res
        # 也可以把主体逻辑写在类外，会快很多
```

```python
class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        # 先初始化一个表格
        dp = [[inf] * (amount + 1) for _ in range(len(coins) + 1)]
        dp[0][0]=0
        # 第0行表示没有硬币面额了，所以需要从第1行开始
        for i in range(1,len(coins)+1):
            coin=coins[i-1] # 比如当前是第1行，则当前面额是第0个，索引错开了
            for j in range(amount+1):
                # 默认不选
                dp[i][j]=dp[i-1][j]
                # 选的情况下，需要与不选比较
                if j>=coin:
                    dp[i][j]=min(dp[i][j],dp[i][j-coin]+1)
        res=dp[len(coins)][amount]

        if res!=inf:
            return res
        return -1
```

3. 自底而上的空间优化
```python
class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        # 第 i行仅仅依赖第 i-1 行自身，不需要存储整张二维表格，完全可以只用一维数组滚动更新。
        dp=[inf]*(amount+1) # dp[i]表示凑够i金额需要的最少硬币数
        dp[0]=0 # 边界条件
        # 外层循环：遍历硬币面额
        for c in coins:
            # 内层循环，遍历从c到amount的剩余金额
            for j in range(c,amount+1): # 对于j小于当前c的情况，在当前c进行处理没有任何意义，因为一定是inf
                dp[j]=min(dp[j],dp[j-c]+1)
        res=dp[amount]
        return res if res!=inf else -1
```
