### 题目描述
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？


### 思路
动态规划
状态转移方程：dfs(i)=dfs(i-1)+dfs(i-2)

但是纯裸递归会疯狂重复计算：
比如算 `dfs(5)` 要算 `dfs(4)`+`dfs(3)`；
`dfs(4)` 又要再算一遍 `dfs(3)`+`dfs(2)`，大量重复递归分支，时间指数爆炸。
可以加入缓存装饰器，第一次算出 dfs (i) 的结果就存进哈希表，下次再调用同参数直接查表返回，不再递归重复计算


### 代码
```python
class Solution:

    def climbStairs(self, n: int) -> int:

        def dfs(i):
            if i<=1:
                return 1
            return dfs(i-1)+dfs(i-2)
        return dfs(n)
```
加装饰器
```python
class Solution:

    def climbStairs(self, n: int) -> int:

        @cache
        def dfs(i):
            if i<=1:
                return 1
            return dfs(i-1)+dfs(i-2)
        return dfs(n)
```
自下而上的思想
```python
class Solution:

    def climbStairs(self, n: int) -> int:

        f=[0]*(n+1)
        f[0]=f[1]=1

        for i in range(2,n+1):
            f[i]=f[i-1]+f[i-2]
            
        return f[n]
```
空间优化，对于i只需要知道i-1和i-2的状态即可
```python
class Solution:

    def climbStairs(self, n: int) -> int:

        f0=1
        f1=1

        for _ in range(2,n+1):
            new_f=f0+f1
            f0=f1
            f1=new_f
            
        return f1
```
