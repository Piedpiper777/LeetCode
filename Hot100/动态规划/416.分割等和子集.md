### 题目描述
给你一个 **只包含正整数** 的 **非空** 数组 `nums` 。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。


### 思路

设nums的和为s
如果s是奇数，直接False
如果s是偶数，则题目是使两个子集的和为s/2
所以这题就转化为了，能否从nums中找到一个子集，使其和为s/2，就成了0-1背包问题

### 代码
1. 自上而下
```python
class Solution:

    def canPartition(self, nums: List[int]) -> bool:

        @cache
        def dfs(i,j):
            # 剩余容量为j，第i个元素选还是不选

            # 边界条件，所有元素遍历完毕
            if i<0:
                if j==0:
                    return True
                else:
                    return False

            # 递归主体
            if nums[i]>j:
                return dfs(i-1,j)
            return dfs(i-1,j) or dfs(i-1,j-nums[i])

        s=sum(nums)

        if s%2!=0:
            return False

        return dfs(len(nums)-1, s//2)
```

2. 自下而上
```python
class Solution:

    def canPartition(self, nums: List[int]) -> bool:

        s=sum(nums)

        if s%2!=0:
            return False

        dp=[[False]*(s//2+1) for _ in range(len(nums)+1)]

        # 边界条件
        dp[0][0]=True

        for i in range(1,len(nums)+1):
            num=nums[i-1]

            for j in range(s//2+1):
            
                dp[i][j]=dp[i-1][j]
                
                if j>=num:
                    dp[i][j]=dp[i][j] or dp[i-1][j-num]

        return dp[len(nums)][s//2]
```

```python
class Solution:

    def canPartition(self, nums: List[int]) -> bool:

        s=sum(nums)

        if s%2!=0:
            return False

        dp=[False]*(s//2+1)
        dp[0]=True

        for num in nums:
            for j in range(s//2,num-1,-1):
                dp[j]=dp[j] or dp[j-num]

        return dp[s//2]
```
