### 题目描述
给你一个字符串 `s` 和一个字符串列表 `wordDict` 作为字典。如果可以利用字典中出现的一个或多个单词拼接出 `s` 则返回 `true`。

注意：不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。



### 思路
考虑切片s[:i]能否返回True，可以拆解为s[j:i]和s[:j]能都返回True，而j则需要枚举，从i-1一直枚举下去，这里有两个终点：可以是-1，也可以是对字典中最长的字符串做差，即i-maxlen-1

### 代码
1. 自上而下
```python
class Solution:

    def wordBreak(self, s: str, wordDict: List[str]) -> bool:

        words=set(wordDict)
		@cache
        def dfs(i):
            """
            表示切片s[:i]是否能返回True
            """

            # 边界条件
            if i==0:
                return True
            # 枚举
            for j in range(i-1,-1,-1):
                if s[j:i] in words and dfs(j):
                    return True
            return False

        return dfs(len(s))
```
2. 自下而上
```python
class Solution:

    def wordBreak(self, s: str, wordDict: List[str]) -> bool:

        max_len=max(map(len,wordDict))
        words=set(wordDict)

        n=len(s)
        f=[True]+[False]*n

        # 第一个True是边界
        for  i in range (1,n+1):
            for j in range(i-1,-1,-1):
                if f[j] and s[j:i] in words:
                    f[i]=True
                    break
        return f[n]
```
