### 题目描述
给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号 子串 的长度。
左右括号匹配，即每个左括号都有对应的右括号将其闭合的字符串是格式正确的，比如 `"(()())"`

### 思路
首先我们很熟悉怎么消括号，这种题已经做过了，现在只需要把消掉的括号标注一下，统计一下标注过的最长长度即可


### 代码
1. 思路一
```python
class Solution:

    def longestValidParentheses(self, s: str) -> int:
        n=len(s)

        # 初始化一个列表记录该位置的括号是否被配对
        is_valid=[False]*n

        # 初始化一个栈，记录左括号
        stack=[]

        for i,x in enumerate(s):

            if x=='(':
                stack.append(i)

            # 不是左括号就是右括号，但是需要栈非空
            elif stack:
                is_valid[i]=True
                is_valid[stack.pop()]=True

        res=0
        cnt=0
        for i in range(n):
            if is_valid[i]:
                cnt+=1
                res=max(res,cnt)
            else:
                cnt=0
        return res
```

2. 思路二，空间优化
```python
class Solution:

    def function(self,s,left_ch):
        res=0
        # 分别统计左右括号的个数
        left=0
        right=0

        for x in s:
            if x==left_ch:
                left+=1

            else:
                right+=1

            if left<right:
                left=right=0

            elif left==right:
                res=max(res,right*2)

        return res

    def longestValidParentheses(self, s: str) -> int:
        return max(self.function(s,'('),self.function(reversed(s),')'))
```
