### 题目描述
给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 _该数组中和为 `k` 的子数组的个数_ 。
子数组是数组中元素的连续非空序列。

### 思路
首先，这题默认没有排序的，双指针需要排序
前缀和的思想
前缀和s[i]表示从0到i的元素的和，则如果s[i]-s[j] == k，则表示从j+1到i的连续子数组满足要求，进而转换为寻找i之前的前缀和中，s[j]=s[i]-k的个数
因此需要统计每一步前缀和的次数，用哈希字典
### 代码
```python
class Solution:

    def subarraySum(self, nums: List[int], k: int) -> int:

        cnt = defaultdict(int)
        ans = s = 0

        #  s是前缀和，包括自己
        #  转换为s[i]-s[j]==k，则从j+1到 i的连续子数组即满足要求
        #  进一步转换为s[j]=s[i]-k，即当前前缀和与k的差值，在前面出现过的次数

        for x in nums:
            cnt[s] += 1
            s += x
            ans += cnt[s - k]

        return ans
```
