### 题目描述
给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。

返回 _滑动窗口中的最大值_ 。

### 思路
暴力解法就是统计每个滑动窗口内的最大值

维护一个递减的队列，左端是该窗口的最大值，右端是最小值
对于每一个右端新进的元素，只要队列中的旧元素比它小就通通弹出不再考虑
同时控制队首，如果队首索引已经出窗口，则弹出


### 代码
```python
class Solution:

    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:

        # 初始化双端队列
        deque=collections.deque() # 存索引下标而不存数值
        
        res=[]

        n=len(nums)

        # l表示左边界，范围是(1-k, n-k+1)
        # i表示右边界，范围是(0,n-1)

        for i,x in enumerate(nums):
        
            # 右端进队列
            # 维持递减，左大右小
            while deque and nums[deque[-1]] < x:
                deque.pop()
            deque.append(i)

            # 左端出队列
            l=i-k+1
            if deque[0]<l:
                deque.popleft()

            if l>=0:
                res.append(nums[deque[0]])
                
        return res
```
