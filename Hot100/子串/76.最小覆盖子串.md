### 题目描述
给定两个字符串 `s` 和 `t`，长度分别是 `m` 和 `n`，返回 s 中的 **最短窗口 子串**，使得该子串包含 `t` 中的每一个字符（**包括重复字符**）。如果没有这样的子串，返回空字符串 `""`。

测试用例保证答案唯一。

### 思路
与209题类似，滑动窗口，满足条件时比较，然后缩小左端边界


### 代码
```python
class Solution:

    def minWindow(self, s: str, t: str) -> str:

        cnt_t=Counter(t)
        cnt_s=Counter()

        ans_l=-1
        ans_r=float('inf')

        l=0
        
        for r,x in enumerate(s):
            cnt_s[x]+=1
            while cnt_s>=cnt_t:
                if r-l<ans_r-ans_l:
                    ans_l,ans_r = l,r
                cnt_s[s[l]]-=1
                l+=1

        if ans_l<0:
            return ""
        return s[ans_l:ans_r+1]
```

优化版
```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        # 统计t中每个字符出现的次数
        diff=defaultdict(int)
        for c in t:
            diff[c]-=1
        # 统计t中字符种类
        kinds=len(diff)

        ans_left=-1
        ans_right=float('inf')

        cnt=0 # 统计s的子串覆盖掉t中字符的种类，每完全覆盖掉一个就加一
        left=0

        for right,c in enumerate(s):
            diff[c]+=1
            if diff[c]==0:
                cnt+=1
            
            # 完全覆盖掉时就是备选答案，开始比较
            while cnt==kinds:
                if right-left<ans_right-ans_left:
                    ans_left,ans_right=left,right
                x=s[left]

                # 缩小左端窗口
                if diff[x]==0:
                    cnt-=1
                diff[x]-=1
                left+=1
        return "" if ans_left<0 else s[ans_left:ans_right+1]
```
