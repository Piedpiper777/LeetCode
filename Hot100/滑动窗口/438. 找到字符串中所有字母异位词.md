### 题目描述
给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。


### 思路
统计p中各个字符出现的次数，找到一个则对应-1，所有都为零则找到了

1. 定长滑动窗口
窗口大小为p的长度len(p)，r-l+1=len(p)，右端点是遍历的，左端点l=r-len(p)+1

2. 不定长滑动窗口
不再单独维护窗口内每个字符出现的次数。右端点进入窗口时，p的次数-1，如果该字符的次数为负，说明该字符多了，从左边缩小窗口，直到没有次数再为负
如果窗口大小等于p的长度，则说明找到了
### 代码
1. 定长窗口
```python
class Solution:

    def findAnagrams(self, s: str, p: str) -> List[int]:

        res=[]

        # 1. 统计p的各字符数量
        cnt_p=Counter(p)

        # 2. 统计窗口内各个字符出现额次数
        cnt_s=Counter()

        # 3.右指针和字符元素
        for r,x in enumerate(s):

            # 右边界加入统计
            cnt_s[x]+=1

            # 确定左边界
            l=r-len(p)+1
            if l<0:
                continue

            # 判断挂窗口是否是
            if cnt_s==cnt_p:
                res.append(l)

            # 左边界出窗口
            cnt_s[s[l]]-=1

        return res
```

2. 不定长
```python
class Solution:

    def findAnagrams(self, s: str, p: str) -> List[int]:

        # 1. 统计p中字符出现次数
        cnt_p=Counter(p)

        res=[]

        l=0

        for r,x in enumerate(s):

            # 右端点进入
            cnt_p[x]-=1

            # 如果为负，说明多了，缩小窗口
            while cnt_p[x]<0:
                cnt_p[s[l]]+=1
                l+=1

            # 如果窗口大小和p长度相等，说明找到了
            if r-l+1 == len(p):
                res.append(l)
                
        return res
```
