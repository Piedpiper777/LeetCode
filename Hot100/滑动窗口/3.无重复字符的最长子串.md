### 题目描述

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。
### 思路
1. 滑窗
新建一个字典，存储字符元素出现的最新索引位置
用当前元素索引减去字典中存储的该字符上次出现的索引
j是左边界的前一个值，i是右边界
如果dic(s[i])在j左边，则左边界仍是j，不能回退到dic(s[i])
如果dic(s[i])在j右边，则左边界就是dic(s[i])

2. 动态规划
思路类似，以s[i]为结尾的最长字符串的长度有两种可能：以s[i-1]为结尾的长度+1，或者从上一个s[i]元素出现的索引到现在
### 代码
1. 滑窗
```python
class Solution:

    def lengthOfLongestSubstring(self, s: str) -> int:

        dic={}
        res=0
        j=-1

        for i in range(len(s)):

            if s[i] in dic:
                j=max(j,dic[s[i]])

            dic[s[i]]=i

            res=max(res,i-j)

        return res
```
2. 动态规划
```python
class Solution:

    def lengthOfLongestSubstring(self, s: str) -> int:

        res=0
        dic={} # key是字符，value是字符在字符串中的最新索引
        temp=0

        for i in range(len(s)):
            j=dic.get(s[i],-1) # 获取上一个最新的索引，没有出现过则返回-1
            dic[s[i]]=i # 当前的索引

            # 动态规划方法
            if temp<i-j: # j+1到i的长度是i-j
                temp=temp+1

            else:
                temp=i-j
                
            res=max(temp,res)

        return res
```
