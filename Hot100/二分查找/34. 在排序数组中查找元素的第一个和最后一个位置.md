### 题目描述
给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。


### 思路
1. 写两次二分查找
2. 写一次二分查找找到起始点，然后调用二分查找函数找到target+1的插入位置-1即可


### 代码
1. 两次二分查找
```python
class Solution:

    def searchRange(self, nums: List[int], target: int) -> List[int]:

        n=len(nums)

        l1=0
        l2=0
        r1=n-1
        r2=n-1

        while l1<=r1:
            mid=(l1+r1)//2
            if nums[mid]>=target:
                r1=mid-1
            else:
                l1=mid+1

        while l2<=r2:
            mid=(l2+r2)//2
            if nums[mid]>target:
                r2=mid-1
            else:
                l2=mid+1

        if  l1<=n-1 and nums[l1]==target:
            return [l1,r2]
        return [-1,-1]
```
2. 优化方法
```python
class Solution:

    def fun(self,nums,target):

        n=len(nums)
        l=0
        r=n-1

        while l<=r:
            mid=(l+r)//2
            if nums[mid]>=target:
                r=mid-1
            else:
                l=mid+1
        return l

    def searchRange(self, nums: List[int], target: int) -> List[int]:
        start=self.fun(nums,target)
        if start>=len(nums) or nums[start]!=target:
            return[-1,-1]
        end=self.fun(nums,target+1)-1
        return [start,end]
```
