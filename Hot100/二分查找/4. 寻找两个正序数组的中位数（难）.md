### 题目描述
给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。
算法的时间复杂度应该为 O(log (m+n)) 。

### 思路
暴力做法是把两个数组合并然后排序
实际上中位数的作用是把元素按照大小左右分组

假设较短数组长度为n，较长数组长度为m
从较短数组长度入手，从中间切开
i+j应该满足等于（m+n+1)//2
### 代码
```python
class Solution:

    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:

        if len(nums1) > len(nums2):
            nums1,nums2=nums2,nums1

        # m是较短数组，n是较长数组
        m=len(nums1)
        n=len(nums2)
        
        l=0
        r=m
        
        while l<=r:
            i=(l+r)//2
            j=(m+n+1)//2-i
            # 短数组左边是[0:i]
            # 常数组左边是[0,j]

            ## 边界条件
            ### 如果i=0，则短数组左边是空的，即L1不存在
            ### 如果i=m，则短数组右边是空的，即R1不存在
            ### 如果j=0，则长数组左边是空的
            ### 如果j=n，则长数组右边是空的

            if i==0:
                L1=float('-inf')
            else:
                L1=nums1[i-1]

            if i==m:
                R1=float('inf')
            else:
                R1=nums1[i]

            if j==0:
                L2=float('-inf')
            else:
                L2=nums2[j-1]

            if j==n:
                R2=float('inf')
            else:
                R2=nums2[j]

            if L1>R2:
                # 说明短数组左边拿多了，需要缩小，缩小r
                r=i-1
            elif L2>R1:
                # 说明短数组左边拿少了，需要增大，增大l
                l=i+1
            else:
                # 已经满足条件了
                if (m+n)%2==1:
                    # 长度是奇数，取左边最大值
                    return max(L1,L2)
                else:
                    # 长度是偶数，取左边最大值和右边最小值的平均值
                    return (max(L1,L2)+min(R1,R2))/2.0
```
