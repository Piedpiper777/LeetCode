### 题目描述
给你一个满足下述两条属性的 `m x n` 整数矩阵：

- 每行中的整数从左到右按**非严格递增**顺序排列。
- 每行的第一个整数大于前一行的最后一个整数。

给你一个整数 `target` ，如果 `target` 在矩阵中，返回 `true` ；否则，返回 `false` 。


### 思路
1. 笨方法
先找到再哪一行，再简化为单行的二分查找
2. 优化方法
直接把二维矩阵展平为一维数组，mid=(l+r)//2，将mid转成矩阵中的坐标：[mid//n][mid % n]
### 代码
1. 笨方法
```python
class Solution:

    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:

        m=len(matrix)
        n=len(matrix[0])
        i=0

        if target<matrix[0][0] or target>matrix[m-1][n-1]:
            return False

        while i<m and target>matrix[i][n-1]:
            i+=1

        l=0
        r=n-1

        while l<=r:
            mid=(l+r)//2
            if target>matrix[i][mid]:
                l=mid+1
            else:
                r=mid-1

        if target==matrix[i][l]:
            return True

        return False
```
2. 优化方法

```python
class Solution:

    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:

        m=len(matrix)
        n=len(matrix[0])

        l=0
        r=m*n-1

        while l<=r:
            mid=(l+r)//2
            x=matrix[mid//n][mid%n]

            if x==target:
                return True

            elif x>=target:
                r=mid-1

            else:
                l=mid+1

        return False
```
