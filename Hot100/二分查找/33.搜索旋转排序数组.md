### 题目描述
整数数组 `nums` 按升序排列，数组中的值 **互不相同** 。

在传递给函数之前，`nums` 在预先未知的某个下标 `k`（`0 <= k < nums.length`）上进行了 **向左旋转**，使数组变为 `[nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]]`（下标 **从 0 开始** 计数）。例如， `[0,1,2,4,5,6,7]` 下标 `3` 上向左旋转后可能变为 `[4,5,6,7,0,1,2]` 。

给你 **旋转后** 的数组 `nums` 和一个整数 `target` ，如果 `nums` 中存在这个目标值 `target` ，则返回它的下标，否则返回 `-1` 。

你必须设计一个时间复杂度为 `O(log n)` 的算法解决此问题。


### 思路
首先得找到旋转点，或者说原数组的最小值或者最大值，然后二分查找
1. 写两次二分查找
2. 写一次二分查找
让x=nums[mid]，如果target>nums[-1]>x，说明target落在第一段，x落在第二段，则缩小右边界，反之缩小左边界，否则的话就是二者落在同一段

### 代码
1. 两次二分
```python
class Solution:

    def findk(self,nums):
        """这里寻找最小值"""
        n=len(nums)
        l=0
        r=n-1
        while l<=r:
            mid=(l+r)//2
            if nums[mid]>nums[n-1]:
                # 说明此时中间点落在了左边，缩小左边界
                l=mid+1
            else:
                r=mid-1
        return l

  

    def func(self,nums,target):
        n=len(nums)
        l=0
        r=n-1

        while l<=r:
            mid=(l+r)//2
            if nums[mid]>=target:
                r=mid-1
            else:
                l=mid+1
        return l
        # 如果target在nums中，返回的l就是其索引，如果不在，返回的l是其插入位置的索引

  

    def search(self, nums: List[int], target: int) -> int:
        k=self.findk(nums)
        n=len(nums)
        if nums[n-1]<target:
            i=self.func(nums[0:k],target)
            if nums[i]!=target:
                return -1
            return i
        else:
            j=self.func(nums[k:n],target)
            if nums[k+j]!=target:
                return -1
            return k+j
```

由于这种写法是用切片的方式，切片是新数组，所以需要额外处理索引，可以直接在原数组操作
```python
class Solution:

    def findk(self,nums):
        """这里寻找最小值"""
        n=len(nums)
        l=0
        r=n-1
        while l<=r:
            mid=(l+r)//2
            if nums[mid]>nums[n-1]:
                # 说明此时中间点落在了左边，缩小左边界
                l=mid+1
            else:
                r=mid-1
        return l
        
    def func(self,nums,target,l,r):
        while l<=r:
            mid=(l+r)//2
            if nums[mid]>=target:
                r=mid-1
            else:
                l=mid+1
        if nums[l]==target:
            return l
        return -1
        # 如果target在nums中，返回的l就是其索引，如果不在，返回-1

    def search(self, nums: List[int], target: int) -> int:
        k=self.findk(nums)
        n=len(nums)
        if target>nums[n-1]:
            return self.func(nums,target,0,k-1)
        return self.func(nums,target,k,n-1)
```
2. 一次二分查找
```python
class Solution:

    def search(self, nums: List[int], target: int) -> int:
        n=len(nums)
        l=0
        r=n-1
        
        while l<=r:
            mid=(l+r)//2
            if target>nums[n-1]>=nums[mid]:
                r=mid-1
            elif nums[mid]>nums[n-1]>=target:
                l=mid+1
            elif nums[mid]>=target:
                r=mid-1
            else:
                l=mid+1
        if l<n and target==nums[l]:
            return l
        return -1
```
