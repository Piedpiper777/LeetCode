### 题目描述
给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 `O(log n)` 的算法。


### 思路
分治法
左指针为0，右指针为n-1，计算中间指针mid=(l+r)//2，比较nums[mid]和target的大小，如果nums[mid]>target，说明target落在[l:mid]，令r=mid-1，如果nums[mid]< target，让l=mid+1。
不断缩小右边界，可以找到第一个target

### 代码
```python
class Solution:

    def searchInsert(self, nums: List[int], target: int) -> int:

        n=len(nums)
        l=0
        r=n-1

        while l<=r:

            mid=(l+r)//2

            if nums[mid]<target:
                l=mid+1

            else:
                r=mid-1

        return l
```
