### 题目描述
给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。


### 思路

思路一：统计每个元素出现次数，然后遍历，但是占用了额外空间

思路二：摩尔投票
### 代码
1. 思路一
```python
class Solution:

    def majorityElement(self, nums: List[int]) -> int:

        cnt=Counter(nums)

        for k,v in cnt.items():

            if v>len(nums)//2:

                return k
```

2. 思路二，摩尔投票
```python
class Solution:

    def majorityElement(self, nums: List[int]) -> int:

        res=nums[0]
        cnt=1

        for x in nums[1:]:
            if x==res:
                cnt+=1

            else:
                cnt-=1
                
                if cnt==0:
                    res=x
                    cnt=1

        return res
```
