### 题目描述
给你一个整数数组 `nums` ，数组中的元素 **互不相同** 。返回该数组所有可能的子集（幂集）。
解集 **不能** 包含重复的子集。你可以按 **任意顺序** 返回解集。


### 思路
1. 选还是不选的思想
2. 选哪个的思想
### 代码
1. 选不选
```python
class Solution:

    def subsets(self, nums: List[int]) -> List[List[int]]:

        def dfs(x:int):

            if x==len(nums):
                res.append(list(path))
                return

            dfs(x+1)

            path.append(nums[x])
            dfs(x+1)
            path.pop()

        path=[]
        res=[]
        dfs(0)
        
        return res
```

2. 选哪个
```python
class Solution:

    def subsets(self, nums: List[int]) -> List[List[int]]:

        def dfs(x:int):
            res.append(list(path))

            for j in range(x,len(nums)):
            
                path.append(nums[j])

                dfs(j+1)

                path.pop()

        res=[]
        path=[]
        dfs(0)
        
        return res
```
