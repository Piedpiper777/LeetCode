### 题目描述
按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

**n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n×n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n` ，返回所有不同的 **n 皇后问题** 的解决方案。

每一种解法包含一个不同的 **n 皇后问题** 的棋子放置方案，该方案中 `'Q'` 和 `'.'` 分别代表了皇后和空位。

### 思路
其实是行列两层遍历，不过我们可以手动控制r的遍历，或者说其实是锁定了，因为皇后必定一行一个，当然控制列也可以。
这样只需再操作每一行的列即可
合法性要求：行列索引之差和之和不能相等，只要有一个相等就不合法


### 代码

```python
class Solution:

    def solveNQueens(self, n: int) -> List[List[str]]:

        res=[]
        col_location=[0]*n  # queen放在(r,col_location[r])
        col_is=[False]*n # 表示该列是否已经用过了
        diag1=[False]*(n*2-1) # 表示行列索引之和是否已存在
        diag2=[False]*(n*2-1) # 表示行列索引之差是否已存在

        # 当前该放第r行了
        def dfs(r):
            # 终止条件，超出索引边界
            if r==n:
                res.append(['.'*c+'Q'+'.'*(n-1-c) for c in col_location])
                return

            # 递归主体，如果列没用过，并且也满足和差要求，则可以防止，然后递归下一个
            for c,ok in enumerate(col_is):
                if not ok and not diag1[r+c] and not diag2[r-c]:
                    col_location[r]=c
                    col_is[c]=diag1[r+c]=diag2[r-c]=True
                    dfs(r+1)
                    col_is[c]=diag1[r+c]=diag2[r-c]=False

        dfs(0)

        return res
```
