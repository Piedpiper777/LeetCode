### 题目描述
数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。


### 思路
1. 选和不选
合法要求：已经填的右括号的个数不能大于已经填的左括号的个数
所以当前操作有两种：如果左括号还没填到n就可以填左括号；如果右括号的个数小于左括号则可以填右括号

```python
class Solution:

    def generateParenthesis(self, n: int) -> List[str]:

        res=[]
        path=[]

        # l表示已经填的左括号的个数，r表示已经填的右括号的个数
        def dfs(l,r):
            # 终止条件
            if r==n:
                res.append(''.join(path))

            # 递归主体
            # 只要左括号还有个数就可以填
            if l<n:
                path.append('(')
                dfs(l+1,r)
                path.pop()
            # 如果右括号个数＜左括号个数，也可以填
            if r<l:
                path.append(')')
                dfs(l,r+1)
                path.pop()

        dfs(0, 0)

        return res
```
