### 题目描述
给定一个 `m x n` 二维字符网格 `board` 和一个字符串单词 `word` 。如果 `word` 存在于网格中，返回 `true` ；否则，返回 `false` 。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

### 代码
```python
class Solution:

    def exist(self, board: List[List[str]], word: str) -> bool:
		# 优化一：如果board中的字符数量＜word，直接不可能
        cnt = Counter(c for row in board for c in row)
        if not cnt >= Counter(word):
            return False

        # 优化二：如果word第一个字母出现的次数大于最后一个字母出现的次数，反转word
        if cnt[word[-1]]<cnt[word[0]]:
            word[::-1]

        m=len(board)
        n=len(board[0])

        # i,j表示当前格子坐标，k表示当前匹配到的word的字母，即下一步要在board[i][j]匹配word[k]

        def dfs(i,j,k):

            # 如果当前格子不匹配，直接返回False
            if board[i][j]!=word[k]:
                return False

            # 没有返回Flase说明匹配成功，如果当前k已经是最后一个了，那就返回True
            if k==len(word)-1:
                return True

            # 递归主体
            # 1.为了避免重复访问，需要把当前格子清掉
            board[i][j]=''

            # 2.搜索到了
            for x,y in (i+1,j),(i-1,j),(i,j+1),(i,j-1):
                # 要规定新索引的边界
                if 0<=x<m and 0<=y<n and dfs(x,y,k+1):
                    return True
            # 恢复现场
            board[i][j]=word[k]

            # 3.没搜到
            return False

        # 这里不能直接在循环里面return res，如果是True的话可以直接return，但是如果是Flase只能表明当前起点不行
        # 需要所有起点都False才可以返回Flase
        for i in range(m):
            for j in range(n):
                res=dfs(i,j,0)
                if res:
                    return True

        return False
```
