### 题目描述
给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。答案可以按 **任意顺序** 返回。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

### 思路
首先肯定需要见建一个映射表

回溯思想：当前操作是什么？子问题是什么？下一个子问题是什么？

### 代码
```python
Mapping=['', '', 'abc', 'def', 'ghi', 'jkl', 'mno', 'pqrs', 'tuv', 'wxyz']

class Solution:

    def letterCombinations(self, digits: str) -> List[str]:

        n=len(digits)

        if n==0:
            return []

        path=['']*n
        res=[]

        def dfs(x):
			# 如果超出索引时，直接返回
            if x==n:
                res.append(''.join(path))
                return
			# digits是一个字符串，而digits[x]表示其中的索引为x的字符，套int是因为原本是str
			# Mapping[int(digits[x])]表示索引为x的字符对应的字母组合
			# 对其实中的每一个字符进行遍历，先加进来，然后再看下一个
            for c in Mapping[int(digits[x])]:
                path[x]=c
                dfs(x+1)
                
        dfs(0)

        return res
```
