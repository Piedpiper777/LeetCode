### 题目
给定一个不含重复数字的数组 `nums` ，返回其 _所有可能的全排列_ 。你可以 **按任意顺序** 返回答案。

### 思路
1. 子集类问题
用一个是否用过列表来控制
2. 交换
固定当前位置，让当前位置元素和后面某一元素交换，然后递归后续元素
### 代码
1. 子集类问题
```python
class Solution:

    def permute(self, nums: List[int]) -> List[List[int]]:

        n=len(nums)
        path=[0]*n
        res=[]
        used=[False]*n # 表示第i个元素是否已经被列入了

        def dfs(i):

            # 终止条件：边界
            if i==n:
                res.append(list(path))
                return

            # 递归主体
            # 遍历使用列表
            for j,u in enumerate(used):
                # 如果第j个元素还没有列入，只有没有用过的才考虑后续列入
                if not u:
                    path[i]=nums[j] # 把没用过的元素列入
                    used[j]=True # 已经用过了
                    dfs(i+1) # i已经解决了，考虑子问题
                    used[j]=False # 弹出，为下一次遍历腾空间

        dfs(0)

        return res
```
2. 交换
```python
class Solution:

    def permute(self, nums: List[int]) -> List[List[int]]:

        def dfs(x:int):

            if x==len(nums)-1:
                res.append(list(nums))
                return
                
            for i in range(x,len(nums)):
                nums[x],nums[i]=nums[i],nums[x]
                dfs(x+1)
                nums[x],nums[i]=nums[i],nums[x]

        res=[]
        dfs(0)
        return res
```
