### 题目描述
给你一个字符串 `s`，请你将 `s` 分割成一些 子串，使每个子串都是 **回文串** 。返回 `s` 所有可能的分割方案。

### 思路
1. 选还是不选
但是这题选与不选的对象是每个字符后面是否要切开
2. 枚举法
这一题的枚举相对容易理解，即假设现在的起点是i，则枚举i后面的索引

### 代码
1. 选还是不选
```python
class Solution:

    def partition(self, s: str) -> List[List[str]]:

        n=len(s)
        path=[]
        res=[]

        # i表示s中的当前位置，start表示当前字符串的起始位置
        def dfs(i,start):

            if i==n:
                res.append(list(path))
                return

            # 判断是否要在i之后分割

            # 不分割（但是i=n-1时必须分割所以需要加一个判断）
            if i<n-1:
                dfs(i+1,start)

            # 分割
            t=s[start:i+1]
            # 判断当前切下来的这一串是不是回文串
            if t==t[::-1]:
                path.append(t)
                dfs(i+1,i+1)
                path.pop()

        dfs(0,0)

        return res
```

2. 枚举法
```python
class Solution:

    def partition(self, s: str) -> List[List[str]]:

        n=len(s)
        path=[]
        res=[]

        def dfs(i):
            # 终止条件，超出索引边界，即所有元素都考虑进来了无法再切了
            if i==n:
                res.append(list(path))
                return

            for j in range(i,n):
                t=s[i:j+1] # 表示索引从i到j的字符串
                if t==t[::-1]:
                    path.append(t)
                    dfs(j+1)
                    path.pop()

        dfs(0)

        return res
```
