### 题目描述
给你一个链表数组，每个链表都已经按升序排列。
请你将所有链表合并到一个升序链表中，返回合并后的链表。

### 思路
（1）思路一：列表排序
把所有节点放在一起，排序然后构建新列表，这里有两种方法，存节点和存值
（2）思路二：最小堆
（3）思路三：分治法（递归+迭代）


### 代码
- 存节点
```python
class Solution:

    def mergeKLists(self, lists):
        l=[] # 初始化一个空列表
        # 把每个链表中的每个元素都存进去
        # 1.外循环，遍历每个链表
        for x in lists:
	        # 2.内循环，遍历链表中的每个元素
            cur=x
            while cur:
                l.append(cur)
                cur=cur.next
        # 3.排序，按照值的大小排序
        l.sort(key=lambda x: x.val)
		# 4.构建新列表
        pre=ListNode(0)
        res=pre
        for y in l:
            pre.next=y
            pre=pre.next
        return res.next
```

- 最小堆
```python
ListNode.__lt__ = lambda a, b: a.val < b.val  
class Solution:
    def mergeKLists(self, lists):
        h=[]
        for head in lists:
            if head:
                h.append(head)

        heapify(h)

        cur=dum=ListNode(0)

        while h:
            node=heappop(h)
            cur.next=node
            cur=cur.next
            if node.next:
                heappush(h,node.next)

        return dum.next
```

- 分治-递归（自上而下）
```python
class Solution:

    def TwoListsMerge(self, list1:Optional[ListNode], list2: Optional[ListNode]) ->Optional[ListNode]:
        cur=dum=ListNode(0)
        while list1 and list2:
            if list1.val<list2.val:
                cur.next=list1
                list1=list1.next
            else:
                cur.next=list2
                list2=list2.next
            cur=cur.next
        if list1:
            cur.next=list1
        if list2:
            cur.next=list2
        return dum.next

    def mergeKLists(self, lists):    
        m=len(lists)
        if m==0:
            return None
        if m==1:
            return lists[0]
        left=self.mergeKLists(lists[:m//2])
        right=self.mergeKLists(lists[m//2:])
        return self.TwoListsMerge(left,right)
```
