### 题目描述
给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。
请你设计并实现时间复杂度为 O(n) 的算法解决此问题。

### 思路
先去重
动态规划：以nums[i]结尾的最长连续序列=（以nums[i-1]结尾的最长连续序列+1 or 1）
当然也可以不用动态规划

### 代码
```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        set1=set(nums)
        res=0
        for x in set1:
            if x-1 in set1:
                continue
            y=x+1
            while y in set1:
                y+=1
            res=max(y-x,res)
        return res
```

```python 
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        max_len = 0
        s=set(nums)
        for x in s:
            if x-1 not in s:
                cur_num=x
                cur_len=1
                while cur_num+1 in s:
                    cur_num+=1
                    cur_len+=1
                max_len=max(max_len,cur_len)
        return max_len
```

```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        dp=dict()
        max_len = 0
        s=set(nums)
  
        for x in s:
            if x-1 not in s:
                cur=x
                while cur in s:
                    dp[cur]=dp.get(cur-1,0)+1
                    max_len=max(max_len,dp[cur])
                    cur+=1
        return max_len
```
